FourD

Web Component: Force Directed Graph Visualization

Installation

Install global prerequisites:

  • npm install -g parcel
  • npm install -g mocha

First-time project setup:

  • git clone https://github.com/stauffenbits/fourd
  • cd fourd
  • npm install

You can repeat this process (deleting the fourd folder in between) to update to newer versions of this library, or simply run

  • git pull origin main

Usage

Thereafter, to build the project and see it live at http://localhost:1234

  • npm run local

Open your browser at http://localhost:1234 and open your browser’s dev tools.


Here is a program to get you started:

var v1 = fourd.add_vertex(), v2 = fourd.add_vertex(), v3 = fourd.add_vertex(), v4 = fourd.add_vertex(); fourd.add_edge(v1, v2); fourd.add_edge(v2, v3); fourd.add_edge(v1, v3); fourd.add_edge(v1, v4); fourd.add_edge(v2, v4); fourd.add_edge(v3, v4); fourd.follow(1);

Once an improvement is ready to share

  • npm run publish

which will stage, commit, and upload your files to github, prompting you for your username and repo password. (I think they deprecated plain password authentication, simply head over to settings and set up an app password).
